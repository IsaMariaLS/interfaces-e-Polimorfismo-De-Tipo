# Atividade: Interfaces e Polimorfismo (Figuras 2D)
Nesta atividade, foi feito o uso de interfaces para cálculos matemáticos e ordenação de coleções.

O que foi implementado:
- Interface FiguraGeometrica2D com métodos para area(), perimetro() e tipoFigura().
- Figuras: Implementações de Circulo e Quadrado.
- Ordenação: A interface estende Comparable, permitindo que o RepositorioDeFiguras2D ordene a lista da menor para a maior área usando Collections.sort().
